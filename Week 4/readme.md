# Week 4 - HTML Basics

This folder contains basic HTML practice assignments.

## Topics Covered

- HTML document structure
- Headings and paragraphs
- Links using `<a>`
- Images using `<img>`
- Ordered and unordered lists
- Nested lists
- Inline styling
- Basic profile page layout

## Files

| File | Description |
|---|---|
| `Asg1.html` | Basic webpage using paragraphs, heading, Google link, unordered lists, and nested lists. |
| `Asg2.html` | Cat profile page with heading, image, introduction, ordered list, useful links, background color, and horizontal line. |

## How to Run

Open any HTML file in a browser:

```text
Asg1.html
Asg2.html
```

## Summary

Week 4 focuses on learning basic HTML tags and creating simple static web pages.
