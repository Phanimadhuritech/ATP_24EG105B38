# Week 4 - HTML and CSS Practice

This folder contains Week 4 practice work for HTML structure, basic CSS styling, tables, cards, grid layout, and full webpage layouts.

## Topics Covered

- HTML document structure
- Headings, paragraphs, links, and images
- Ordered, unordered, and nested lists
- Inline styling and external CSS files
- Profile page layout
- Card design using CSS
- HTML tables with `rowspan` and `colspan`
- CSS Grid layout
- Navigation bars and multi-section webpage layouts
- Button links using anchor tags

## Folder Structure

| Folder | Files | Description |
|---|---|---|
| `BASIC WEB PAGE` | `Basic html .html`, `Pochi_the_cat.html` | Basic HTML pages with paragraphs, headings, Google link, lists, cat profile, images, useful links, and inline background styling. |

| `HTML` | `grid.html`, `grid.css` | CSS Grid practice layout with multiple grid child sections. |
| `FULL WEB PAGE` | `Asg1.html`, `Asg1.css`, `index.html`, `styles.css` | Full webpage layouts including a Vampire Diaries card page with Read More links and a Libertarian-style layout with hero section and columns. |
